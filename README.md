# todo-list-sederhana
Project pertama menggunakan javascript.. Mungkin terlihat simpel tapi aku percaya "practice makes perfect"

## Fitur
- **Tambah tugas:** Masukan tugas dan tekan enter atau tombol tambah.
- **Coret/Ceklis Tugas:** Klik tugas untuk menandainya sebagai selesai.
- **Hapus Tugas:** Klik tombol silang (X) untuk menghapus tugas selamanya.
- **Penyimpanan Otomatis:** Data tersimpan di LocalStorage browser, jadi tidak hilang saat di-refresh.
- **Dark Mode:** Ganti tema terang/gelap.

## Teknologi yang Digunakan
- HTML5
- CSS3 (Flexbox & CSS Variables)
- Vanilla JavaScript

## Cara Menjalankan
1. Download atau Clone repository ini.
2. Buka file `index.html` di browser (Chrome, Firefox, Edge).
3. Selamat menggunakan!
